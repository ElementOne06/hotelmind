# HotelMind MVP

HotelMind es una aplicación inicial para hoteles pequeños y medianos.

Esta primera versión permite visualizar:

- Ingresos totales
- Gastos totales
- Utilidad estimada
- Ocupación
- ADR
- RevPAR
- Habitaciones
- Reservaciones
- Recomendaciones básicas

## Ejecutar localmente

```bash
streamlit run app.py
